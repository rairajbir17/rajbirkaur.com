Getting started with Prestashop

PrestaShop is an open source application which enables you to create your e-commerce site quickly, free of charge. Download Prestashop, install the software and start selling online.

This guide is mainly target to people who are wishing to SELL their store products online.

System Requirements:

Following is the list of system requirements need to get started with the installation of Prestashop.

  • System Requirements:
    • Php 5.4 or later
    • MySQL 5.0 or later
    • Good if
      • Unix/Linux hosting.
      • Apache Web Server 2.0 or later.
      • Minimum 128 Mb RAM.
